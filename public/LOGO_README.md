# Logo Setup

To add your logo to the dashboard:

1. Place your PNG logo file in the `public` folder
2. Rename it to `logo.png`

The logo will automatically appear in the header of the dashboard.

**Recommended logo specifications:**
- Format: PNG (transparent background recommended)
- Height: 40-60px
- Width: Auto (maintains aspect ratio)
- Background: Transparent or matching your brand

The logo is styled with a light blue glow effect to match the dashboard theme.
